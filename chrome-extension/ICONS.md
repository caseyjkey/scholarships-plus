# Icons Needed

The extension needs the following icon sizes:

- **icon16.png** (16x16) - Toolbar icon
- **icon48.png** (48x48) - Extension management page
- **icon128.png** (128x128) - Chrome Web Store, installation dialog

## Icon Design

**Style:** Modern, clean gradient with sparkle emoji

**Colors:**
- Primary: Blue gradient (#3b82f6 → #8b5cf6)
- Accent: White sparkle emoji (✨)

**Design:**
- Rounded square with subtle shadow
- Sparkle emoji (✨) prominently displayed
- Gradient background: Blue to purple
- White sparkle emoji centered

## Temporary Placeholder

For development, you can:
1. Use any placeholder icons
2. Create simple PNG files with an online tool
3. Use the Chrome extension icon generator: https://icon.kitchen

## Production Icons

When ready for Chrome Web Store:
- Generate professional icons at all required sizes
- Ensure icons follow Chrome Web Store design guidelines
- Include 128x128 for store listing
